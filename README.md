# WebBlog - ASP.NET Core Blog System

## Description
WebBlog is a simple blog management web application developed using ASP.NET Core MVC.  
The system allows administrators to manage blog posts and content while enabling users to browse and read articles.

## Technologies Used
- ASP.NET Core MVC
- C#
- SQL Server
- Entity Framework Core
- Razor Views

## Features
- Create, edit, and delete blog posts
- Display blog post lists and detailed article pages
- Category-based blog browsing
- Database integration using SQL Server
- MVC architecture for clean separation of logic and UI

## System Architecture
- Backend: ASP.NET Core MVC
- Database: SQL Server
- ORM: Entity Framework Core

## Installation

1. Clone the repository

git clone https://github.com/ShanYangk14/WebBlog


2. Open the project in Visual Studio

3. Configure the SQL Server connection string

4. Run the application

dotnet run
